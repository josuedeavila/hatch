# hatch
Write agent rules and commands once and generate for all.

1. Copy the .hatch folder into your repo
2. Edit the .hatch/commands and .hatch rules
3. Run ./hatch/gen.sh to generate config

**Supported IDEs:**

- Aider
- Claude Code
- Cody
- Continue
- Cursor
- GitHub Copilot
- VS Code
- Windsurf
- Zed
